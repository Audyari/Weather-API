# Simple Express.js Backend

Backend Express.js sederhana dengan Hello World endpoint.

## Instalasi

1. Install dependencies:

```bash
npm install
```

2. Jalankan server:

```bash
npm start
```

Atau untuk development mode:

```bash
npm run dev
```

## Endpoints

- `GET /` - Hello World
- `GET /health` - Health check

Server akan berjalan di http://localhost:3000
